## Hello World

Ask questions to the user and get answers.

## What you will learn in this project:
- `scanf` is a part of `stdio.h` library
- `scanf()` allows us to take input from user

## Some other points:

An example `scanf` function looks like this:
```c
#include <stdio.h>

int main() {
	int age;
	printf("What is your name");
	scanf("%d", &age);
	printf("Your age is %c", age);
}
```
